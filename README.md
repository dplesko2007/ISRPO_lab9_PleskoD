### Лабораторная работа №9

**Описание работы:** Во время выполнения данной лабораторной работы были изучены принципы работы с `@media`, также масштабирование сайтов для разных разрешений устройств.

**Файла и папки**:
* `img` - папка со скриншотами выполнения заданий лабораторной работы.
* `index.html` - файл для рассмотрения принципов работы *@media*
* `style.css` - файл для рассмотрения принципов работы *@media*
* `responsive.html` и `responsive.css` - парные файлы. Практический пример работы масштабирования.
* `nav-practice.html` и `nav-pratcie.css` - парные файлы, самостоятельное выполнение и применеие *@media*.
* `flexbox-practice.html` и `flexbox-pratice.css` - парные файлы, самостоятельное выполнение и применеие *@media*.
* `README.md` - описание проделанной работы.
